This is a Log Search and alerting tool.
Below are the features of this project :
1. Designed for scalability-Multiple process can be started and offload log server
2. Remote Monitoring of the process with Web app
3. Realtime adding of keywords-no server stop
4. To off load the log server-Active MQ is used
5. Scalable DB to store the Alerts-Mongo DB
6. Web UI to show the alerts
7. Web UI to monitor the alerts
8. Different alerts such as Email,SMS or posting to Twitter/Crowaround
